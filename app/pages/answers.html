1. Что такое EcmaScript?

Это собственно язык программирования, на основе которого реализован JS.

2. Для чего используются и чем отличаются атрибуты async и defer?

Оба значения позволяют асинхронно загружать код, отличие в том, что async выполнит код сазу как только его загрузит, а defer будет подгружать скрипты асинхронно, но выполнит их в том порядке, в которм они установлены.

3. Почему предпочтительнее подключать скрипты перед , а не в ...?

Станица загружается сверху вниз, поэтому если в начале будет "тяжелый" скрипт, то будет долго грузиться страница. Нас же интересует, чтобы мы сперва увидели контент сайта, а потом, уже пускай начинают работать скрипты (например тяжелые анимации)

4. Какие основные отличия интерпретируемого языка программирования от компилируемого?

Интерпретируемый выполняет код "построчно" и прекращает его выполнение, если где-то будет ошибка, но все что до этого -- сработает. Компилируемый сперва проверит код на правильность и не выполнит его вовсе, если будет там ошибка.

5. Что такое языки высокого и низкого уровня? Какие ты знаешь?

Низкоуровневый язык программирования -- язык с использованием команд машинного кода, т.е. команд определенного вычеслительного процессора предназначеной для конкретной машины. Обычно используется путем мнемонических обозначений (буквенных или условных), которые машина переводит в двоичный код. Пример: группа языков Ассамлера. Высокоуровневый язык програмирования -- работают с использованием смысловых консрукций, что определяют работу кода. Они (конструкции) кратко описывают данные, с которыми работает программист, и действия, что производятся над ними.

6. Для чего используются переменные? Какие ключевые слова используются для создания переменных? Какие есть отличия в переопределении значений при задании переменной через let, const и var?

Переменные используются для хранения информации с которой в дальнейшем мы будем работать. Ключевые слова -- var, let и const. Переменная var видна везде в функции (т.е. она сработает, если будет указана после функции) и ее можно переопределить; let -- имеет область видимости { }, она будет "существовать" только если объявлена ДО функции, ее можно переназначить; const -- аналогична let, но ее нельзя переназначить и как правило она постоянна (?).

7. Может ли имя переменной начинаться с цифры, символов _ или $?

Да, может.Кроме цифры. С цифры названия не должны начинаться. Также нельзя использовать "-".

8. Какие слова запрещено использовать в названии переменной?

Те, которые используются самим языком программирования: "return, class, var, const, for, do, in" и др. Список зарезервированных слов всегда можно посмотреть: https://msdn.microsoft.com/ru-ru/library/0779sbks(v=vs.94).aspx

9. Что такое camelCase и snake_case?

Это способ написания для имени переменной.

10. Что такое "типы данных"? Какой оператор позволяет определить тип данных?

Оператор "typof" позволяет определить тип данных. Всего их шесть: number, string, Boolean, null, undefined, object. То есть это, соответсвенно числа (целые и дробные); "строки"; логические значения (true & false); null === ничего/неизвестно; значение не указано; объекты, т.е. то, что содержит каки-либо данные (это как павило {...} и [...])

11. Что такое "динамическое приведение типов данных"? Можно ли писать JavaScript со статическим приведением типов?

Динамическая типизация -- то, что работает по дефолту. Т.е. js-интерпритатор сам понимает какой тип данных мы ему передаем: const a = "string" // typeof string const b = 64 // typeof number const c = true // typeof Boolean Да, можно писать js-код со статистическим приведением типов с помощью TypeScript. Для обычного программиста в этом нет необходимости и только усложнит ему работу, но может использоваться для разработки каких-либо приложений, игр и т.д.

12. Как привести строчное значение "10" к числу? Назовите два способа.

1) Поставить в начале "+". Например: var num = "10"; console.log(+num) // 10 // typeof number 2) Использовать в начале "Number": var num = "10" console.log(Number(num))

13. Чем отличаются операторы "==" и "==="?

"==" сравнивает значение, "===" сравнивает также тип данных: 22 == "22" // true 22 === "22" // false

14. Какие преимущества дают обратные кавычки (`) при указании строки?

1) При использовании обратных кавычек возможен перенос текста на новую строку; 2) При помощи ${...} можно внутри строки задавать какие-либо значения: const age = 30 console.log(`Мне сегодня ${age} лет`) // Мне сегодня 30 лет
